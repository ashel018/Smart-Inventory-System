# Smart Inventory System 

A Java-based console application for managing product inventory and orders using a MySQL database.

## Features
- **Add Products:** Register new items with price and quantity.
- **Manage Stock:** Update and delete existing products.
- **Order System:** Place orders and track sales.
- **Database Integration:** Persistent storage using JDBC and MySQL.

## Setup Instructions
1. Clone this repository.
2. Create a MySQL database named `inventory_db`.
3. Update the credentials in `src/com/inventory/DBConnection.java`.
4. Add the MySQL Connector JAR to your project dependencies.
5. Run `Main.java`.
